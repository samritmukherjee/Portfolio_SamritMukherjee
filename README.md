# 🖥️ SAMRIT OS

> **A portfolio experience that's actually an operating system**

You don't visit SAMRIT OS. You *boot* it. 

This isn't another portfolio website built with Next.js. This isn't a collection of links and PDFs. This is a fully functional, interactive operating system—complete with dual-boot, draggable windows, working applications, and authentic OS animations—living in your browser.

Pick your environment. Choose macOS Ventura or Windows 11. Drag windows around. Minimize, maximize, organize. Explore applications. Experience actual desktop computing. All while discovering who I am as a developer.

## ⚙️ Under the Hood

Built with modern web technologies to feel and perform like a desktop operating system:

```
Frontend:        Next.js 14 + React 18 + TypeScript
Styling:         Tailwind CSS
Animation:       Framer Motion (60fps optimized)
State:           React Context API
Window System:   react-rnd (draggable & resizable)
OS Simulation:   Custom-built macOS & Windows shells
```

The entire UI is a custom desktop environment—nothing imported from OS libraries, everything purpose-built for this experience.
